Bloom by The WombA Professional Clinical Pregnancy CompanionBuilt by an OB-GYN for expectant mothers, Bloom is designed to bridge the gap between clinic visits. It combines "cute," supportive UI with rigorous clinical monitoring for high-risk pregnancies (PIH & GDM).🌟 Key FeaturesPrecision POG Tracker: Calculates Period of Gestation (POG) down to the day using LMP.The Lotus Nest: Interactive dashboard with animated fruit size comparisons (e.g., Avocado at Week 16).High-Risk Clinical Modules:7-Point GDM Charting: Specialized logging for Gestational Diabetes (Fasting, Post-prandial, 2 AM).Bi-Daily PIH Monitor: Blood pressure tracking with automated clinical alerts for readings $\ge 140/90$ mmHg.Fetal Movement Protocol: Integrated "Kick Counter" with emergency instructions for decreased movement.The Village: A community support hub and direct 24/7 doctor chat interface.Digital Scrapbook: Upload scans and reports to generate an automated Antenatal PDF record.🛠️ Tech StackLanguage: Python 3.9+Framework: StreamlitData Handling: PandasPDF Generation: ReportLabVisualization: Plotly (for BP and Glucose trends)
